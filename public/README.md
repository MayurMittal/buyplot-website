# Public Assets Directory

## Founder Photo

**IMPORTANT:** Please add the founder's photo here as `founder-photo.jpg`.

### File Location
Place the image file directly in this `public` folder with the exact name: `founder-photo.jpg`

### Supported Formats
The component will try to load:
- `founder-photo.jpg` (primary)
- `founder-photo.png` (fallback)
- `founder-photo.jpeg` (fallback)
- `founder-photo.webp` (fallback)

### Image Requirements
- **Recommended size:** 800x1000px or larger (portrait orientation)
- **Aspect ratio:** 4:3 or similar works well
- **Format:** JPG, PNG, JPEG, or WebP
- **File size:** Optimize for web (under 500KB recommended)

### How to Add the Image

1. Save your founder's photo/image
2. Rename it to `founder-photo.jpg` (or one of the supported formats)
3. Copy it to this `public` folder
4. The image will automatically appear on the website

### Current Status
If the image doesn't load, a fallback placeholder with initials will be displayed.
